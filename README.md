# Bangladesh Rainfall Prediction using Machine Learning

This project applies multiple Machine Learning algorithms to analyze a Bangladesh weather dataset and predict rainfall.

## Dataset
The dataset contains weather-related features used to predict the **rain** variable.

Features include weather measurements collected over time in Bangladesh.

## Project Workflow
1. Load and explore the dataset
2. Data preprocessing
3. Train-test split
4. Feature scaling using StandardScaler
5. Training multiple machine learning models
6. Evaluating model performance

## Machine Learning Models Used

- Linear Regression
- Ridge Regression
- Lasso Regression
- Decision Tree
- Random Forest
- Gradient Boosting
- Extra Trees
- K-Nearest Neighbors (KNN)
- Support Vector Regression (SVR)
- XGBoost

## Libraries Used

- Python
- NumPy
- Pandas
- Scikit-learn
- XGBoost

## Goal

The goal of this project is to compare different machine learning models and identify which model performs best for predicting rainfall based on weather data.
